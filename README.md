# pretalx-schedule

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Build for pretalx (web component)
```
npm run build:wc
```

Then copy ``dist/*js`` to ``src/pretalx/static/agenda/js/`` in pretalx.

### Lints and fixes files
```
npm run lint
```
